応募・選考について

Q 応募資格はありますか？ +
４年生大学を卒業していること
Q 応募方法は？ +
弊社採用サイトからエントリーしてください。
Q 選考フローは？ +
書類選考ののち数回の面接を予定しております。
Q 選考にかかる時間は？ +
1週間ほどお時間を頂いております
Q 面接では何を聞かれますか？ +
志望動機などお聞かせいただきます
Q 面接官はどんな人ですか？ +
弊社社員にて行います

キャリアについて

Q 自分の希望するキャリアパスに進むことはできますか？ +
個人の能力を考慮いたします。
Q 転勤はありますか？ +
業務の必要に応じて、転勤が発生する場合があります。
Q 転勤の希望は出せますか？ +
可能です。ご相談ください。
Q ワークライフバランスは？ +
福利厚生などの様々な制度があります。
Q 具体的にはどのような制度がありますか？ +
保養所や財形など他にも多数ご用意しています。
Q 育児休暇制度はありますか？ +
あります。リフレッシュ休暇や産前産後休暇など取得可能です。

福利厚生について

Q どのような福利厚生制度がありますか？ +
社員向けジムや社会保険など完備しております。
Q 財形貯蓄制度の内容は？ +
財形貯蓄を支援する制度です。
Q 社内イベントの内容は？ +
スポーツ大会などレクリエーションを行います
Q 部活動について教えてください。 +
テニス部、サッカー部、野球部があります。社員であればどなたでも入部可能です。
Q 資格取得支援制度はいくら補助が出ますか？ +
ファイナンシャルプランナー2級取得時に1万円支給します。
Q 年次休暇について教えてください +
12/29-1/3までの6日間
